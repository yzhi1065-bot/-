# 中医智能诊断系统 — UI/UX 界面设计方案

版本：V1.0  
日期：2026年6月23日

---

## 1. 设计原则

### 1.1 核心设计理念

| 原则 | 说明 |
|------|------|
| **医师为本** | 遵循中医师诊疗流程习惯，不做颠覆性改变 |
| **极简高效** | 单次诊疗点击 < 5次完成核心操作 |
| **数据可视化** | 脉象图谱、舌诊影像等直观呈现 |
| **AI透明可信** | AI建议来源可追溯、可解释、可修改 |
| **中医美学** | 界面融入中国传统文化元素，配色温润 |

### 1.2 色彩体系

```
主色调：    #8B4513 (檀棕色) — 稳重、传统、专业
辅助色：    #D4A574 (杏色)   — 温暖、亲和
强调色：    #C0392B (朱砂红) — 重要提示、诊断结论
背景色：    #F5F0EB (宣纸白) — 柔和、护眼
文字主色：  #2C2C2C           — 高可读性
文字辅色：  #8C8C8C           — 次要信息
成功/健康： #27AE60 (草药绿)
警告：      #E67E22
```

### 1.3 字体

- **中文**：Noto Sans SC / 思源黑体
- **英文/数字**：Inter / JetBrains Mono
- **字号**：正文14px，次要12px，标题16-20px

## 6. 交互细节

### 6.1 四诊采集步骤指引

```
步骤01 ─━ 望诊    当前
步骤02 ── 闻诊    待完成
步骤03 ── 问诊    待完成
步骤04 ── 切诊    待完成

完成度: ████████░░ 75%
```

- 顶部 Step Bar 显示采集进度
- 每个步骤可跳转（已完成的可以回退修改）
- 设备连接失败时，弹出重试/手动录入选项

### 6.2 脉波图可视化

- 实时绘制脉波曲线（Canvas/SVG）
- 标注关键参数：脉率、脉幅、脉宽、脉形
- 支持缩放、导出图谱
- 显示参考正常波形对比

### 6.3 舌诊影像标注

- 自动舌体分割（高亮边界）
- 舌质/舌苔区域自动划分
- AI标注关键特征（裂纹、齿痕、芒刺等）
- 医生可手动修正标注

### 6.4 快捷键设计

| 快捷键 | 功能 |
|--------|------|
| F1 | 开始新诊疗 |
| F2 | 采集确认/下一步 |
| F3 | 生成AI诊断 |
| F5 | 刷新设备连接 |
| Ctrl+S | 保存当前进度 |
| Ctrl+P | 打印报告 |
| Esc | 返回/取消 |

## 7. 深色模式（可选）

- 支持浅色/深色模式切换
- 深色模式采用低对比度设计，减少眼部疲劳
- 医疗影像类内容在深色模式下保持原色
