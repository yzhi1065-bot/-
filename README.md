# 中医智能诊断系统

> **采集-分析-诊断-治疗** 一体化的中医AI辅助诊断平台

一个完整的中医诊所数字化解决方案。连接硬件设备采集四诊数据（望闻问切），调用AI大模型进行辨证论治，辅助中医师完成诊断和治疗方案制定。

## 🚀 快速开始

### 环境要求
- Python 3.11+
- Node.js 18+
- npm 9+

### Windows

```bash
# 一键启动
双击 start.bat

# 或手动启动
cd backend
pip install -r requirements.txt
python run.py

# 新终端
cd frontend
npm install
npm run dev
```

### Mac / Linux

```bash
chmod +x start.sh
./start.sh
```

### 访问

```
前端界面: http://localhost:3000
后端API:  http://localhost:8000
API文档:  http://localhost:8000/api/docs
```

## 🌐 部署

### Docker 部署

```bash
# 构建前端
cd frontend
npm install && npm run build

# 启动所有服务
docker-compose up -d
```

### 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| DATABASE_URL | sqlite:///./tcm.db | 数据库连接 |
| AI_API_KEY | - | AI模型API密钥 |
| AI_API_URL | http://localhost:8010 | AI服务地址 |
| SECRET_KEY | (生产环境需修改) | JWT密钥 |
